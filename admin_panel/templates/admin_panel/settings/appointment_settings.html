Configuration Help
Max Appointments Per Slot

How many appointments can be booked for the same time slot. Usually 1 for exclusive service.

Cancellation Deadline

Minimum notice required for cancellations. Helps prevent last-minute cancellations.

Auto-confirm

Automatically approve new appointments without manual review. Speeds up booking process.

Employee Assignment

Require appointments to be assigned to specific employees before confirmation.

Reminders

Send automatic reminder notifications to customers before their appointments.

Best Practices
  • Set reasonable cancellation deadlines (24-48 hours)
  • Auto-confirm for faster customer experience
  • Require employee assignment for better scheduling
  • Send reminders 24 hours before appointments
  • Limit slots to prevent overbooking
